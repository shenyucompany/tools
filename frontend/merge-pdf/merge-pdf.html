PDF合并工具

快速合并多个PDF文件，支持批量处理

📖 使用说明

  1. 点击"选择文件夹"按钮，选择包含需要合并的PDF文件的文件夹
  2. 系统会自动识别文件夹中的所有PDF文件并显示在列表中
  3. 确认文件列表无误后，点击"合并PDF"按钮
  4. 等待处理完成后，系统会自动下载合并后的PDF文件

Mac打印设置

macOS

  1. 选择双面打印
  2. 设置页面布局为每页2版
  3. 调整打印方向为纵向

Windows
👆 点击查看大图

设置步骤：

  1. 勾选双面打印选项
  2. 在版式中选择每页2版
  3. 确保打印方向为纵向
